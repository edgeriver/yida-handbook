# 图片下载功能模块

本文档整理了组件中关于图片选择与下载的核心逻辑，包含生命周期初始化、状态管理、单/批量下载实现及UI交互控制。

---

## 组件初始化与依赖加载

在组件挂载阶段，负责检测运行环境并动态加载必要的第三方库。

### 生命周期钩子：`didMount`

该函数在组件加载完成后执行，主要处理以下逻辑：

- **环境校验**
  - 检测当前环境是否为**移动端**或**钉钉工作台**。
  - 若在上述环境中，则禁止加载下载脚本（因为移动端通常不支持文件直接下载）。
- **动态加载脚本**
  - 依次加载 `JSZip`（用于文件压缩）和 `FileSaver.js`（用于触发浏览器下载）。
  - 加载成功后在控制台打印提示信息。
- **异常处理**
  - 若脚本加载失败，通过 `toast` 提示用户“图片下载加载失败”。
- **数据获取**
  - 调用 `getData()` 方法初始化页面数据。

```jsx
export async function didMount() {
  if (!this.utils.isMobile() || !this.isDingTalkEnv()) {
    try {
      await this.utils.loadScript('https://g.alicdn.com/code/lib/jszip/3.9.1/jszip.min.js');
      await this.utils.loadScript('https://g.alicdn.com/code/lib/FileSaver.js/2.0.5/FileSaver.min.js');
      console.log('DownloadImage Load Success');
    } catch (e) {
      this.utils.toast({
        title: '图片下载加载失败',
        type: 'error'
      });
    };
  };
  this.getData(); // 获取数据
}
```

## 图片选择与交互逻辑

### 图片选择回调

当用户在表格中勾选或取消勾选图片时触发，同步更新组件状态。

- 接收两个参数：`selectedRowKeys`（选中行ID）和 `records`（选中行详细数据）。
- 调用 `setState` 更新本地状态，确保 UI 与数据保持一致。

```jsx
export function onImageSelect(selectedRowKeys, records) {
  this.setState({
    selectedImageRowKeys: selectedRowKeys,
    selectedImageDetail: records,
  });
}
```

### 打开下载弹窗

点击“下载”按钮时触发，负责前置校验及弹窗初始化。

**执行流程：**

1. **环境限制检查**
   - 若在钉钉或移动端环境，直接弹出警告提示并终止流程。
2. **数据存在性检查**
   - 若当前行数据中不存在图片字段 (`imageField_l9nvelqm`)，提示“暂无图片需要下载”。
3. **初始化弹窗内容**
   - 显示弹窗 (`dialog_legisiyq`)。
   - 自动生成带时间戳的默认文件名（如：`批量下载文件_20260101123045`）。
   - 将图片数据注入到弹窗内的表格中。
   - 清空之前的选中状态，防止数据残留。

```jsx
export function onDownloadImageDialog(rowData) {
  const { imageField_l9nvelqm } = rowData;
  if (this.utils.isMobile() || this.isDingTalkEnv()) {
    this.utils.toast({
      title: '不支持在钉钉工作台或移动端中下载',
      type: 'warning',
    });
    return;
  };
  if (!imageField_l9nvelqm) {
    this.utils.toast({
      title: '暂无图片需要下载',
      type: 'warning',
    });
    return;
  }
  this.$('dialog_legisiyq').show(() => {
    this.$('textField_legtgpx9').setValue(`批量下载文件_${this.utils.formatter('date', Date.now(), 'YYYYMMDDhhmmss')}`);
    this.$('tablePc_legpq38q').set('data', JSON.parse(imageField_l9nvelqm));
    this.setState({
      selectedImageRowKeys: [],
      selectedImageDetail: [],
    });
  });
}
```

## 下载核心功能实现

### 单张图片下载

通过创建 `<a>` 标签并模拟点击的方式触发浏览器下载。

- 创建一个临时的 `<a>` DOM 元素。
- 设置 `href` 为图片的下载地址。
- 执行 `click()` 事件。
- 清理 DOM 节点，避免内存泄漏。

```jsx
export function downloadFile(downloadUrl) {
  const link = document.createElement('a');
  link.href = downloadUrl;
  document.body.appendChild(link);
  link.click();
  document.body.removeChild(link);
}
```

### 批量图片打包下载 (ZIP)

利用 `JSZip` 将多张图片打包为一个 ZIP 文件后下载。

**实现步骤：**

1. **遍历图片列表**：对选中的每张图片发起请求获取 Blob 数据。
2. **Promise 聚合**：使用 `Promise.all` 等待所有图片资源加载完毕。
3. **生成 ZIP**：调用 `zip.generateAsync()` 生成 Blob 格式的压缩包。
4. **保存文件**：通过 `saveAs` 函数触发下载。

```jsx
export function downloadFiles(downloadfiles, fileName) {
  const zip = new JSZip();
  const result = downloadfiles.map(item => {
    let promise = getFileBlob(item.downloadUrl).then((res) => {
      zip.file(item.name, res, { binary: true });
    });
    return promise;
  });
  Promise.all(result).then(() => {
    zip.generateAsync({ type: "blob" }).then((res) => {
      saveAs(res, `${fileName}.zip`);
    })
  })
}
```

#### 辅助函数：获取文件 Blob

通过 `XMLHttpRequest` 请求图片资源并返回 Blob 格式数据。

```jsx
function getFileBlob(url) {
  return new Promise((resolve, reject) => {
    let request = new XMLHttpRequest()
    request.open("GET", url, true)
    request.responseType = "blob"
    request.onload = (res) => {
      if (res.target.status == 200) {
        resolve(res.target.response)
      } else {
        reject(res)
      }
    }
    request.send()
  })
}
```

## 确认下载逻辑

在下载弹窗中点击“确定”按钮时执行的最终校验与分发逻辑。

**决策树：**

1. **空选校验**
   - 若未选中任何图片，弹出警告提示并退出。
2. **批量下载 (>1 张)**
   - 校验文件名输入框是否为空。
   - 调用 `downloadFiles` 方法进行打包下载。
3. **单张下载 (=1 张)**
   - 直接调用 `downloadFile` 方法下载当前图片。
4. **关闭弹窗**
   - 无论哪种方式，最终都会隐藏下载弹窗。

```jsx
export function onDownloadImageOk() {
  const { selectedImageDetail = [] } = this.state;
  if (!selectedImageDetail.length) {
    this.utils.toast({
      title: '尚未选择任何图片',
      type: 'warning'
    });
    return;
  };
  if (selectedImageDetail.length > 1) {
    // 批量下载
    this.$('textField_legtgpx9').validate();
    const fileName = this.$('textField_legtgpx9').getValue();
    if (!fileName) { return; };
    this.downloadFiles(selectedImageDetail, fileName);
    this.$('dialog_legisiyq').hide();
  } else {
    // 单张下载
    this.downloadFile(selectedImageDetail[0].downloadUrl);
    this.$('dialog_legisiyq').hide();
  };
}
```
