# 数据加载与交互

本文档详细梳理了从状态变量配置、数据源定义到核心业务逻辑实现的完整流程，旨在帮助开发者快速构建具备分页、排序、搜索功能的表格页面。

## 状态变量配置 (State)

在数据源设置中添加以下变量，用于控制页面的整体状态。所有变量均可通过 `this.state.xxx`进行访问和更新。

|变量名|类型|初始值|作用说明|
|---|---|---|---|
|`pageSize`|number|`10`|每页显示的记录条数|
|`currentPage`|number|`1`|当前页码|
|`searchFieldData`|object|`{}`|存储高级查询条件（高级版专用）|
|`dynamicOrderData`|object|`{}`|存储排序字段，格式如 `{ "textField_l9m5jvma": "-" }`（`-`表示降序，`+`表示升序）|
|`tableIsLoading`|boolean|`TRUE`|控制表格的加载状态（loading）|
|`tableData`|object|`{ data: [], totalCount: 0, currentPage: 1 }`|表格数据源及分页信息|
|`selectedTableDataRowKeys`|array|`[]`|（高级版）批量操作时记录选中行的 `formInstId`|

## 动作面板核心逻辑

### 页面挂载与数据初始化

页面加载完成后，自动触发数据获取逻辑。

```jsx
export function didMount() {
  this.getData();
}
```

### 主数据获取函数 (getData)

该函数负责调用远程接口、处理返回数据并更新 State，是表格渲染的核心驱动力。

```jsx
export function getData() {
  // 从 state 中解构所需参数
  const { pageSize, currentPage, searchFieldData = {}, dynamicOrderData = {} } = this.state;
  
  // 调用数据源接口
  this.dataSourceMap.getData.load({
    formUuid: 'FORM-KW766OD1AGUFQQHJ80EG66IGJ6S13GIUKYROL2', // 替换为实际的 formUuid
    pageSize,
    currentPage,
    searchFieldJson: JSON.stringify(searchFieldData),  // 高级查询条件
    dynamicOrder: JSON.stringify(dynamicOrderData),    // 排序条件
  }).then((res) => {
    // 处理返回数据，解构 formData 以便表格直接使用
    const { totalCount, data = [] } = res;
    const result = data.map((item) => {
      const { formInstId, formUuid, formData = {} } = item;
      return {
        formInstId,   // 实例ID，用于删除、修改
        formUuid,     // 表单ID，用于跳转详情
        ...formData,  // 展开所有表单字段值
      };
    });
    
    // 更新 state，触发表格自动刷新
    this.setState({
      tableData: {
        currentPage,
        data: result,
        totalCount,
      },
      tableIsLoading: false,
    });
  }).catch(({ message }) => {
    // 错误处理
    this.utils.toast({ title: message, type: 'error' });
    this.setState({ tableIsLoading: false });
  });
}
```

## 表格与查询控件配置

### 表格属性绑定

在表格组件中，将以下属性与 State 或函数进行绑定：

|配置项|绑定目标|说明|
|---|---|---|
|**数据源**​|`state.tableData`|表格渲染的数据源|
|**加载状态**​|`state.tableIsLoading`|控制 loading 动画|
|**已选中行**​|`state.selectedTableDataRowKeys`|双向绑定选中行|
|**分页大小**​|`state.pageSize`|每页条数|
|**选择变动回调**​|`onTableDataSelectChange()`|勾选行时触发|
|**分页/搜索/排序触发**​|`onFetchData()`|表格内置操作触发|

### 查询控件事件映射

针对高级查询组件，配置以下事件映射：

|路径|绑定函数|
|---|---|
|查询 > 高级 > 提交时|`onSearch()`|
|查询 > 高级 > 重置时|`onReset()`|

## 辅助函数与业务逻辑

### 表格交互事件处理

```jsx
// 表格行选择变动回调
export function onTableDataSelectChange(selectedRowKeys, records) {
  this.setState({
    selectedTableDataRowKeys: selectedRowKeys,
  });
}

// 分页、排序触发时的统一处理函数
export function onFetchData(params) {
  const { pageSize } = this.state;
  // 如果是搜索触发或页大小改变，强制回到第一页
  if (params.from === 'search' || params.pageSize !== pageSize) {
    params.currentPage = 1;
  }
  
  // 转换排序格式
  const orderTypeText = { desc: '-', asc: '+' };
  let dynamicOrderData = {};
  if (params.orderColumn) {
    dynamicOrderData[params.orderColumn] = orderTypeText[params.orderType];
  }

  this.setState({
    currentPage: params.currentPage,
    pageSize: params.pageSize,
    dynamicOrderData,
    tableIsLoading: true,
  });
  this.getData();
}
```

### 搜索与重置逻辑

```jsx
// 搜索按钮点击事件
export function onSearch(values) {
  this.setState({
    currentPage: 1, // 搜索必须重置到第一页
    searchFieldData: {
      textField_l9m5jvma: values.textField_lorywk26, // 活动名称
      radioField_l9m5jvmc: values.selectField_los2eeiz, // 活动类型
      radioField_l9m5jvmg: values.selectField_l9xx3dnz, // 所属组织
      radioField_l9m5jvmn: values.selectField_lorywk27, // 活动进度
      // 日期范围处理：结束时间设为当天 23:59:59
      dateField_l9m5jvmi: values.cascadeDateField_l9njsj95 
        ? [values.cascadeDateField_l9njsj95.start, new Date(values.cascadeDateField_l9njsj95.end).setHours(23, 59, 59, 999)] 
        : '', 
      // 成员字段格式化
      employeeField_l9m5jvmp: this.formatEmployeeFieldValue(values.employeeField_l9njsj96),
    },
    tableIsLoading: true,
  });
  this.getData();
}

// 重置按钮点击事件
export function onReset(values) {
  this.setState({
    currentPage: 1,
    searchFieldData: {},
    tableIsLoading: true,
  });
  this.getData();
}
```

### 通用工具函数

```jsx
// 判断是否为钉钉容器环境
export function isDingTalkEnv(win) {
  win = win || window;
  return win.navigator && /dingtalk/i.test(win.navigator.userAgent);
}

// 格式化成员字段的值
export function formatEmployeeFieldValue(userData) {
  if (Array.isArray(userData)) {
    return userData.length ? userData.map(item => item.value) : '';
  }
  return userData ? [userData.value] : '';
}
```
