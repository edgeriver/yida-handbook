# 批量删除管理

本文档详细说明了系统中“批量删除活动”功能的配置方式、交互逻辑及后端接口调用细节。

---

## 一、后端接口配置

在执行删除操作前，需预先配置用于提交删除请求的后端接口地址。

| 数据源名称 | 请求地址                                                          |
| :--- |:--------------------------------------------------------------|
| batchDeleteData | \`/${window.pageConfig.appType}/v1/form/deleteFormData.json\` |

> **说明**：`appType` 为动态变量，将根据当前页面配置自动注入。

---

## 二、交互逻辑与流程

该功能涉及两个核心步骤：**打开确认弹窗**与**执行删除操作**。

### 1. 打开批量删除确认弹窗

当用户点击“批量删除”按钮时触发，主要负责前置校验与弹窗初始化。

**执行逻辑：**

- **空选校验**
  - 从组件状态中获取 `selectedTableDataRowKey`（已选中的活动ID列表）。
  - 若列表为空，弹出警告提示“当前未选中任何活动”，并终止流程。
- **显示弹窗**
  - 调用 `show()` 方法打开确认对话框 (`dialog_lpjg5aha`)。
  - 在弹窗内显示当前选中的活动数量，增强用户感知。

```jsx
export function onBatchDeleteActivityDialog(rowData) {
  const { selectedTableDataRowKey = [] } = this.state;
  if (!selectedTableDataRowKey.length) {
    this.utils.toast({
      title: '当前未选中任何活动',
      type: 'warning',
    });
    return;
  }
  this.$('dialog_lpjg5aha').show(() => {
    this.$('numberField_lpjg8ky1').setValue(selectedTableDataRowKey.length);
  });
}
```

### 2. 确认执行批量删除

在弹窗中点击“确定”按钮后触发，负责调用接口并处理响应结果。

**执行流程：**

1.  **状态变更**
    - 将对话框的确认按钮状态设置为 `LOADING`，防止重复提交。
2.  **接口调用**
    - 通过 `dataSourceMap.batchDeleteData.load` 发起请求。
    - 请求体包含表单 UUID、应用类型、待删除的数据实例 ID 列表等关键参数。
3.  **响应处理**
    - **成功时**：
      - 关闭弹窗并显示“批量删除成功”提示。
      - 清空选中状态并重新拉取表格数据，刷新视图。
    - **失败时**：
      - 弹出错误信息提示。
    - **异常捕获**：
      - 捕获网络或系统异常，显示错误提示并重置按钮状态。

```jsx
export function onBatchDeleteActivityOk() {
  const { selectedTableDataRowKeys = [] } = this.state;
  this.$('dialog_lpjg5aha').set('confirmState', 'LOADING'); // 开启对话框加载状态

  this.dataSourceMap.batchDeleteData.load({
    inputs: JSON.stringify({
      body: {
        formUuid: 'FORM-4C833A663C8F4FC7A265CB075A82ED962L0T', // 活动信息表formUuid
        appType: pageConfig.appType, // 应用 appType
        formInstanceIdList: selectedTableDataRowKeys, // 需要删除的数据实例id列表
        asynchronousExecution: true, // 是否需要异步执行该任务
        executeExpression: false, // 是否触发表单校验规则及回调
      },
    }),
  }).then((res) => {
    this.$('dialog_lpjg5aha').set('confirmState', 'NORMAL'); // 关闭对话框加载状态
    const { success, error } = res;
    if (success) {
      this.$('dialog_lpjg5aha').hide();
      this.utils.toast({
        title: '批量删除成功',
        type: 'success',
      });
      setTimeout(() => {
        this.setState({
          tableIsLoading: true,
          selectedTableDataRowKeys: [],
        });
        this.getData(); // 获取数据
      }, 1000);
    } else {
      this.utils.toast({
        title: error,
        type: 'error', // 修正：失败时应为 error 类型
      });
    }
  }).catch(({ message }) => {
    this.utils.toast({
      title: message,
      type: 'error',
    });
    this.$('dialog_lpjg5aha').set('confirmState', 'NORMAL'); // 关闭对话框加载状态
  });
}
```
