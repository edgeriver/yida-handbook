# 对话框填报新记录

本文档介绍如何通过对话框实现表单数据填报与新增记录功能，涵盖**数据源配置**、**弹窗初始化**、**表单校验**及**数据提交**等核心环节。

## 数据源配置

首先需配置后端接口地址，用于获取和提交表格数据。

| 数据源名称      | 请求地址 |
|:-----------| :--- |
| `saveData` | `/${window.pageConfig.appType}/v1/form/saveFormData.json` |

> **注意**：`appType` 变量通常来自全局配置 `pageConfig`，请确保其已正确定义。

## 页面交互实现

### 1. 创建活动弹窗（初始化）

当用户点击“创建活动”按钮时，调用 `onCreateActivityDialog` 方法。此方法负责**显示对话框**并**重置所有表单项**，防止残留旧数据。

```jsx
// 创建活动弹窗
export function onCreateActivityDialog() {
  this.$('dialog_lowmwcke').show(() => {
    // 重置基础信息字段
    this.$('textField_l9njsj8u').reset();      // 活动名称
    this.$('selectField_l9njsj8w').reset();    // 活动类型
    this.$('selectField_l9njsj8v').reset();    // 所属组织

    // 重置时间选择字段
    this.$('dateField_l9njsj8y').reset();      // 开始时间
    this.$('dateField_l9njsj8z').reset();      // 结束时间

    // 重置其他辅助字段
    this.$('selectField_los2eej0').reset();    // 活动进度
    this.$('employeeField_l9njsj90').reset();  // 发起人
    this.$('imageField_l9njsj94').reset();     // 活动图片
  });
}
```

### 2. 确认创建活动（数据新增）

点击对话框“确定”按钮后，将执行 `onCreateActivityOk` 函数。该函数包含**完整的表单校验、数据处理、接口请求及回调处理逻辑**。

#### 核心实现代码

```jsx
// 确认创建活动
export function onCreateActivityOk() {
  // 需要校验的组件唯一标识集合
  const createFieldList = [
    'textField_l9njsj8u',
    'selectField_l9njsj8w',
    'selectField_l9njsj8v',
    'dateField_l9njsj8y',
    'dateField_l9njsj8z',
    'selectField_los2eej0',
    'employeeField_l9njsj90',
  ];

  // 调用表单校验函数
  this.fieldsValidate(createFieldList).then((errorList) => {
    setTimeout(() => {
      // 若校验未通过，则中断提交
      if (errorList.length > 0) {
        return;
      }

      // 开启对话框加载状态，防止重复提交
      this.$('dialog_lowmwcke').set('confirmState', 'LOADING');

      // 组装并提交表单数据
      this.dataSourceMap.saveData.load({
        formUuid: 'FORM-KW766OD1AGUFQQHJ80EG66IGJ6S13GIUKYROL2',
        appType: pageConfig.appType,
        formDataJson: JSON.stringify({
          textField_l9m5jvma: this.$('textField_l9njsj8u').getValue(),       // 活动名称
          radioField_l9m5jvmc: this.$('selectField_l9njsj8w').getValue(),    // 活动类型
          radioField_l9m5jvmg: this.$('selectField_l9njsj8v').getValue(),    // 所属组织
          dateField_l9m5jvmi: this.$('dateField_l9njsj8y').getValue(),       // 开始时间
          dateField_l9m5jvmj: this.$('dateField_l9njsj8z').getValue(),       // 结束时间
          radioField_l9m5jvmn: this.$('selectField_los2eej0').getValue(),    // 活动进度
          employeeField_l9m5jvmp: this.formatEmployeeFieldValue(
            this.$('employeeField_l9njsj90').getValue()
          ),                                                                 // 发起人

          // ⚠️ 图片字段需转换格式以符合接口规范
          imageField_l9nvelqm: (this.$('imageField_l9njsj94').getValue() || []).map((item) => ({
            downloadUrl: item.downloadURL || item.downloadUrl,
            name: item.name,
            previewUrl: item.imgURL || item.imgUrl,
            url: item.url,
          })),
        }),
      })
        .then(() => {
          // 提交成功：关闭弹窗，提示成功，刷新表格
          this.$('dialog_lowmwcke').hide();
          this.utils.toast({ title: '创建成功', type: 'success' });

          setTimeout(() => {
            this.setState({ tableIsLoading: true });
            this.getData();
          }, 1000);
        })
        .catch(({ message }) => {
          // 提交失败：显示错误提示
          this.utils.toast({ title: message, type: 'error' });
        })
        .finally(() => {
          // 无论成功或失败，均关闭加载状态
          this.$('dialog_lowmwcke').set('confirmState', 'NORMAL');
        });
    });
  }, 0);
}
```

### 3. 批量表单校验工具函数

为确保数据完整性，需对指定组件进行校验。

```jsx
// 批量校验指定组件
export async function fieldsValidate(fieldList = []) {
  const result = [];
  for (let i = 0; i < fieldList.length; i++) {
    await this.$(fieldList[i]).validate((errors, values) => {
      if (!errors) return;
      result.push({
        fieldId: fieldList[i],
        errors: this.utils.isMobile() 
          ? errors.errors[fieldList[i]].errors 
          : errors[fieldList[i]].errors
      });
    });
  }
  return result;
}
```

## 关键注意事项

- **字段 ID 必须一致**：代码中使用的组件 ID（如 `textField_l9njsj8u`）需与页面设计器中配置的 ID **完全一致**，否则会导致取值失败。

- **图片字段格式转换**：图片组件返回的数据结构与接口所需参数往往不同，**必须**按示例方式进行映射转换，否则图片数据无法正常提交。

- **加载状态管理**：在接口请求期间，通过 `confirmState` 控制按钮状态（`LOADING` / `NORMAL`），可有效防止重复提交并提升用户体验。
